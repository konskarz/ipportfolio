Editor

Editing articles from the front end. New articles in category Cases will be automatically added to the bottom menu.

Content

  • Title – header title
  • Alias – used in the URL address only for the new article
  • Styles: body, last, h3, h2
  • Table – insert and edit credits-table
  • Gallery – insert gallery
  • Article – insert link to another article
  • Image – insert image, video or presentation

Gallery

previous next
  • Click Gallery, select the folder, e.g. 'sampledata/fruitshop'
  • Aditional dtm-template, e.g. 'sampledata/fruitshop,dtm'
  • Optional add captions, by creating e.g. fruitshop.txt in 'sampledata/fruitshop' with file names and captions separated by '|'

Images

  • Intro Image – bottom menu image
  • Full article image – header image or video
  • Alt text – Aditional CSS formatting of the header
